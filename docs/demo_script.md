# 3分デモ動画スクリプト

YouTube 限定公開用。Zenn 記事に埋め込む。

## 機材・準備

- 画面解像度: 1920 x 1080
- ブラウザは Edge / Chrome、シークレットモード推奨（不要な拡張機能を消す）
- アプリ URL: Azure Container Apps の公開 URL
- 事前に **Critical Demo** をロードしておくと冒頭がスムーズ。動画は何度撮り直しても良い

## 撮影のコツ

- **冒頭5秒で勝負**: Critical の赤いリスクバッジが目に入る瞬間を最初に見せる
- **数値はナレーションで強調**: 視聴者は字幕より声を覚える
- **Agent Reasoning タブ展開時** は少し間を置く（Multi-Agent の構造を見せる時間）
- **音楽は控えめか無音**。デモのテンポを邪魔しない
- 録画後、字幕（日本語）を必ず入れる。審査員がミュートで見ても伝わる
