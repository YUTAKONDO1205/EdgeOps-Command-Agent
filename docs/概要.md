Microsoft Agent Hackathon powered by Tokyo Electron Device

🗓️ 期間: 2026.4.7 - 2026.6.18

主催: クラスメソッド株式会社



協賛: 東京エレクトロンデバイス株式会社



協力: 日本マイクロソフト株式会社



🌟 概要

Zennで初となる企業・個人対抗ハッカソンで、業務改革につながるAgentic AIを作ろう。



いま、AIは「答える」ツールから、自律的に「動く」エージェントへと進化しています。業務現場の課題を解決するアイデアは、大企業のエンジニアだけが持っているわけではありません。現場を知るすべての人が、変革の担い手になれる時代です。



本ハッカソンは今回、個人部門を新たに設け、企業・個人を問わずより幅広い挑戦者を募ります。 業務の「課題」をAIエージェントでいかに鮮やかに解決するか。その問いに向き合うプロジェクトが集まることで、日本の業務現場にたくさんのヒントと、わくわくするような刺激が生まれることを期待しています。



部門

本ハッカソンは2つの部門で開催します。



企業部門

法人に所属する職員によるチーム。業務課題の解決からソリューション開発まで、目的は問いません。



個人部門

所属不問（ただし政府機関等に所属する方を除く）。個人またはチームで参加したい方。



🎁 Azure利用支援

個人部門の方へ

Azureを新規開設すると、Microsoftから200ドル（約3万円相当）の無料クレジットが自動付与されます。



企業部門の方へ

上記の新規開設時の無料クレジットに加えて、Azureの利用をこれから開始される企業様を対象として、1社あたり約5万円相当のAzureクレジットを先着順で提供します。



対象



日本マイクロソフト、東京エレクトロンデバイス、クラスメソッドのいずれともAzureの取引がない企業

参加条件

企業部門: 法人所属の職員による1名以上のチーム。同一企業からの複数チーム参加可。

個人部門: 所属不問。個人またはチームで参加可。



以下の方は参加できません

政府機関等に所属する方 — 賞金の授与が「利益供与」とみなされるリスクがあるため、対象外となります。

企業部門で所属組織の承諾を得ていない方 — 成果物の知的財産権の帰属やシャドーITのリスクを避けるため、参加登録時に「所属組織の承諾済み」のチェックが必須となります。

個人情報の取り扱い

参加者は、主催者、日本マイクロソフト株式会社、および東京エレクトロンデバイス株式会社の3社が、参加者の個人情報（氏名、企業名、メールアドレス等）およびアンケート回答内容を、本ハッカソンの運営および各社の製品・サービスに関する営業・マーケティング活動のために共同で利用することに同意するものとします。
